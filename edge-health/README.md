## 📄 文件说明

| 文件名 | 功能说明 |
|--------|----------|
| `index.html` | 应用主页面，包含HTML结构 |
| `style.css` | 样式文件，定义界面外观 |
| `app.js` | JavaScript脚本，实现动态功能 |
| `esa.config.json` | 阿里云ESA Pages部署配置文件 |

## 🎯 使用说明

1. 打开应用后，首页显示今日健康数据概览
2. 步数卡片显示实时步数和目标进度
3. 卡路里卡片显示今日消耗
4. 睡眠卡片显示睡眠时长
5. 图表区域展示一周步数趋势
6. 数据每3秒自动更新一次
